政府統計による2000年から2015年までの都道府県別人口推計データから、
この間の人口増加率上位・下位何県かを、pythonライプラリのpandasを用いて取得し、
これらの都道府県の人口推移をmatplotlibを用いて可視化し表示するプログラム。
pythonプログラムはpopulation_transition.py
データは05k5-5.xlsx
なお、データは以下のURLからダウンロードした、「人口推計/長期時系列データ 長期時系列データ（平成12年～令和２年）」を加工して用いている。
https://www.e-stat.go.jp/stat-search/files?page=1&layout=datalist&toukei=00200524&tstat=000000090001&cycle=0&tclass1=000000090004&tclass2=000001051180&stat_infid=000013168605&tclass3val=0
